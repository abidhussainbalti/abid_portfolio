Brain Tumor Detection Using Deep Neural Networks

Dates: November 2025 – December 2025

Description / Problem & Solution:
Developed an end-to-end deep learning pipeline to classify brain tumors from MRI scans into four categories: Glioma, Meningioma, Pituitary, and No Tumor. MRI images were preprocessed using resizing, normalization, and tensor conversion. A fully connected deep neural network with ReLU activations was trained for 100 epochs using CrossEntropyLoss and Adam optimizer on GPU. Model performance was evaluated using accuracy, precision, recall, F1-score, confusion matrix, and ROC-AUC.

Tools / Technologies: Python, PyTorch, NumPy, Pandas, Matplotlib

Outcome / Impact: Achieved high accuracy and reliable tumor classification. The pipeline is reproducible and adaptable for similar medical imaging tasks and is ready for integration into a diagnostic aid system.
